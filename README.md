# FaceAttend

> Automated attendance system using face recognition — marks present/absent from a webcam feed into CSV

## What it does
Registers known faces by name from a `/faces` folder. On each run, opens the webcam, detects and recognizes faces in real time using `face_recognition` (dlib), and logs attendance with timestamps to a date-stamped CSV file.

## Quick Start
```bash
git clone https://github.com/yourusername/FaceAttend
cd FaceAttend
pip install -r requirements.txt

# Register faces — add images to faces/ folder:
# faces/Hassan.jpg, faces/Ahmed.jpg, etc.

python register.py        # encode registered faces
python attend.py          # start webcam attendance
```

## Features
- Real-time face detection + recognition from webcam
- Registers faces from JPEG/PNG images in `/faces` folder
- Records: Name, Time, Date, Confidence % to CSV
- Prevents duplicate entries per session
- `--source` flag: webcam index or video file path
- Optional overlay display (bounding boxes + name labels)

## Tech Stack
| Tool | Why |
|------|-----|
| Python 3.11+ | Orchestration |
| `face_recognition` | dlib-based 128D face embeddings |
| `OpenCV` | Webcam capture + frame display |
| `pandas` | CSV attendance log management |
| `numpy` | Embedding distance calculations |

## Architecture
```
FaceAttend/
├── faces/              # Drop student/employee images here
├── encodings.pkl       # Pre-computed face embeddings (generated by register.py)
├── attendance/         # CSV logs by date: 2026-06-08.csv
├── register.py         # Encodes faces from /faces → encodings.pkl
└── attend.py           # Webcam loop: detect → match → log
```

## Requirements
- Python 3.11+
- CMake + dlib (see `dlib` install guide for your OS)
- Webcam
